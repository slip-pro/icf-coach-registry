# G-008+G-009: Vercel Deploy + ICF Brandbook Redesign — Sprint Notes

**Status**: Achieved
**Started**: 2026-04-05
**Completed**: 2026-04-05

## Context

Phase 1 MVP was on GitHub Pages with a temporary URL and neutral gray design.
Owner provided mockup + ICF Brand Guidelines 2025 PDF. Redesign needed to match
official brand colors, typography, and visual elements.

## Key Decisions

- D-007: Vercel for hosting (auto-deploy from GitHub, professional URL)
- D-008: ICF brand colors applied — Deep Blue, Blue, Yellow, Bone palette
- D-009: Nunito as headline font (Google Fonts fallback for Hoss Round)
- D-010: Logo placeholder PNG until designer provides official SVG

## What changed

### Vercel (G-008)
- Connected GitHub repo to Vercel
- Live at: https://icf-coach-registry.vercel.app/
- Auto-deploys on push to main

### Redesign (G-009)
- DESIGN.md rebuilt from ICF Brand Guidelines PDF
- CSS: all custom properties updated to ICF palette
- Header: logo + "Найти коуча ICF Cyprus" with yellow pill badge
- Cards: white default, Blue (#2b379b) hover with white text
- Tags: yellow on hover
- ICF badges: yellow default, white on hover
- Avatar placeholders: white bg + blue border (unified)
- AI button: solid Blue (#2b379b)
- Decorative SVG brush strokes on background
- Nunito for headlines, Plus Jakarta Sans for body

## Owner feedback loop

Multiple iterations based on owner screenshots and references:
1. First attempt (dark hero banner) — rejected, reverted
2. Second attempt (per owner mockup) — approved with adjustments
3. Card hover colors — adjusted from Light Blue to Blue per owner
4. Badge colors — unified to yellow/white per owner
5. Avatar placeholders — unified to white+blue per owner
6. Title font weight — thinned per owner
7. Title split — "ICF Cyprus" in yellow pill, rest plain

## Patterns

- Owner-driven design iteration works better than autonomous design decisions
- Small CSS-only commits + Vercel auto-deploy = fast feedback loop
- Brandbook PDF extraction via PyPDF2 saved significant time
