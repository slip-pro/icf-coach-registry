# G-013: Registration Form Redesign + Bilingual Bio — Sprint Notes

**Status**: Achieved
**Started**: 2026-04-05
**Completed**: 2026-04-05

## Context

Coaches needed to provide bio in multiple languages. Admin needed Status column
visible first in the Google Sheet. Form needed bilingual bio support.

## Key Decisions

- D-010: Bilingual bio — coach chooses 2 languages, catalog shows matching UI language
- D-011: Sheet column reorder — Status first for admin convenience

## What changed

### Google Sheet
- New column order: Status(A), Name(B), Email(C), ICF Level(D), Photo(E), ...
- New columns: Bio 1 Language, Bio 2, Bio 2 Language
- Apps Script fully rewritten for new column positions

### Code
- sheets.js: normalizeLangCode() converts "Russian"→"ru", new bio fields in Coach typedef
- cards.js: getBioForLanguage() picks bio by UI language (fallback: EN → first bio)
- registration.js: two bio blocks with language dropdown + word counter
- i18n.js: new keys for bio language labels
- mock-coaches.json: bilingual bios for testing

## Problems & Solutions

- Bio language mismatch: Sheet had "Russian" but code expected "ru" → added normalizeLangCode()
- Language switch didn't update bio → confirmed re-render works, just needed normalization

## Patterns

- Google Sheet column changes require coordinated update: Sheet + Apps Script + code
- Language normalization needed at data layer (sheets.js), not display layer
