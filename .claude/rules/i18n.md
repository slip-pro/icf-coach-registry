# Internationalization Rules

- All user-facing strings must be translatable (EN / RU / EL)
- Never hardcode UI text — use a translation dictionary/map
- Coach profile content is NOT translated — displayed as the coach wrote it
- Default language: detect from browser locale, fallback to EN
- Language preference stored in localStorage
- Language codes: `en`, `ru`, `el`
